## What is React?

* **React is a JavaScript Library**
* Used to build **interactive User Interfaces (UI)**
* Mainly used for **Web UI**
* Focuses only on the **View layer (Frontend UI)**
* React does **NOT** control the full application flow

üëâ **React = UI only**

## Angular vs React

### React

* JavaScript **Library**
* Lightweight
* Faster UI rendering
* Used mainly for **Frontend**
* Needs external libraries/frameworks

### Angular

* Complete **Framework**
* Heavy
* Slower compared to React
* Controls full application flow
* Built-in tools (routing, forms, HTTP)
